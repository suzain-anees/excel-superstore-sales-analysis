## 📊 Superstore Sales Analysis (Excel Dashboard)

# 🔍 Overview
This project analyzes retail sales data from a Superstore dataset using Microsoft Excel. The goal is to identify key business insights, track sales performance, and understand profit trends across different regions and categories.

# 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Interactive Dashboard  

# 📈 Key Insights
- West region generated the highest sales among all regions  
- Technology category contributed the most to overall revenue  
- Some sub-categories like Tables showed negative profitability  
- Sales show an increasing trend towards the end of the year  
- Discounts have an impact on profit margins  

# 📊 Dashboard Features
- KPI cards (Total Sales, Profit, Orders, Avg Order Value, Return Rate)  
- Region-wise sales analysis  
- Category & sub-category performance  
- Monthly sales trend analysis  
- Top 5 customers by sales  
- Interactive filters (Region, Category, Segment, Year)
